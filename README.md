> *Think tank polímata. Repositório vivo de ideias, projetos, realizações, MVPs e PoCs.*

---

## O que é este repositório

Este vault é o núcleo intelectual e operacional de um ecossistema de 30 anos em construção.  
Aqui vivem — em estado de elaboração permanente — pesquisas, projetos técnicos, obras literárias,  
experimentos filosóficos, protótipos e registros de realizações.

Não é um blog. Não é um portfólio estático.  
É um organismo.

---

## Estrutura

```
00-Inbox/          → captura bruta, ideias em estado larvar
01-Projetos/       → projetos ativos (Hubstry CaaS, GuruDev, Instituto PCIHᶟ)
02-Areas/          → domínios permanentes (pesquisa, financeiro, ecossistema)
03-Recursos/       → templates, referências, frameworks

```

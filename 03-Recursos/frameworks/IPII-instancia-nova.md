# Como instanciar a IPII em novo domínio

## Protocolo em 5 passos

1. **Identificar os sistemas formais** — quais dois (ou mais) sistemas
   com vocabulários distintos precisam se comunicar?

2. **Mapear nos 5 eixos da LexMatrix**
   - i: qual é a categoria ontológica dos tokens?
   - j: qual é o campo do conhecimento / relação semântica?
   - k: qual nível hermenêutico é necessário?
   - t: qual é o regime temporal (execução, vigência, deadline)?
   - l: qual é o paradigma / âmbito alvo?

3. **Definir o banco pan-domínio** — catalogar padrões equivalentes
   entre os sistemas fonte e alvo

4. **Implementar o matching contextual** — heurísticas de seleção
   do "melhor mapeamento" no domínio específico

5. **Documentar como instância** em instances/ do ipii-core
